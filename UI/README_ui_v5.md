# UI v5.0 - 完整集成版本

## 功能说明

### 已集成的功能

1. **数独生成器**
   - 根据难度（简单/中等/困难）自动生成数独题目
   - 使用 `src/generator/sudoku_generator.py`
   - 支持对称挖洞和难度评估

2. **求解算法**
   - **基础DFS算法**: `src/algorithms/solver_basic_v1.py`
     - 简单的深度优先搜索回溯算法
     - 适合理解基本原理
   
   - **AC3+MRV+LCV算法**: `src/algorithms/solver_ac3_mrv_lcv.py`
     - AC-3: 弧一致性约束传播
     - MRV: 最小剩余值启发式
     - LCV: 最少约束值启发式
     - 性能更优，搜索节点更少

3. **性能对比**
   - 可以一键对比所有算法的性能
   - 显示执行时间、搜索节点数、回溯次数

## 使用方法

### 1. 生成数独
- 选择难度（简单/中等/困难）
- 点击"生成数独（按难度）"按钮
- 系统会自动生成符合难度的题目

### 2. 求解数独
- 选择算法（基础DFS或AC3+MRV+LCV）
- 点击"开始求解"按钮
- 查看性能统计面板了解算法表现

### 3. 对比算法
- 点击"对比所有算法"按钮
- 系统会依次运行所有算法并显示对比结果

## 技术架构

```
UI/ui_v5.py (前端界面)
    ↓
src/generator/sudoku_generator.py (生成器)
    ↓
src/algorithms/
    ├── solver_basic_v1.py (基础DFS)
    ├── solver_basic_v2.py (改进DFS)
    ├── solver_mrv_lcv.py (MRV+LCV)
    └── solver_ac3_mrv_lcv.py (AC3+MRV+LCV)
```

## 运行要求

- Python 3.7+
- tkinter (通常随Python安装)
- 确保项目结构完整，src目录在正确位置

## 运行命令

**重要：必须在项目根目录运行！**

```bash
# 在项目根目录（包含 src 和 UI 文件夹的目录）运行
python UI/ui_v5.py
```

如果看到 "✓ 算法和生成器加载成功"，说明导入成功。

### 常见问题

**Q: 显示"数独生成器未加载"？**
A: 确保你在项目根目录运行，不要在 UI 文件夹内运行。

**Q: 如何确认在正确的目录？**
A: 运行 `dir` (Windows) 或 `ls` (Mac/Linux)，应该能看到 `src` 和 `UI` 两个文件夹。
